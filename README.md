Este é um simples programa em C++ que permite realizar o cadastro de usuários, exibir os dados cadastrados e oferece um menu de opções para interação.

Funcionalidades:
Cadastro de Usuários:

Solicita o nome, idade, email e senha do usuário.
Verifica se a idade é superior a 13 anos para permitir o cadastro.
Verifica se o email contém o caractere '@' para ser considerado válido.
A senha deve ter no mínimo 8 caracteres para ser aceita.
Exibição de Dados:

Mostra os dados cadastrados do usuário, incluindo nome, idade e email.
Menu de Opções:

Permite ao usuário escolher entre cadastrar novos dados, exibir os dados cadastrados ou sair do programa.
Como Executar:
Para executar o programa, compile o código em um compilador C++ e siga as instruções fornecidas pelo programa durante a execução.

Instruções de Uso:
Ao selecionar a opção de cadastro, siga as instruções fornecidas para inserir seu nome, idade, email e senha.
Para exibir os dados cadastrados, escolha a opção correspondente no menu.
Para sair do programa, escolha a opção de saída no menu principal.
Requisitos:
Certifique-se de ter um compilador C++ instalado em seu ambiente de desenvolvimento para compilar e executar o programa.

Observações:
Este programa foi desenvolvido para fins educacionais e demonstra o uso básico de estruturas de controle e entrada/saída em C++.
O tratamento de erros para entradas inválidas é limitado neste exemplo, e um desenvolvimento mais robusto seria necessário para um ambiente de produção real.
